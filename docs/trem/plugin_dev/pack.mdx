# 封裝

封裝擴充為 `.trem` 檔案。

## 準備

資料夾看起來應該像下方這樣

```
example/
    info.json     // 擴充基本資訊
    index.js      // 入口點
```

## 方法

建立 數位簽章 `signature.json` 檔案並打包成 `擴充名稱.trem` 檔案

```
npx github:ExpTechTW/TREM-Plugin-Signer sign . && npx github:ExpTechTW/TREM-Plugin-Packer
```

:::warning
需要有數位簽章 `signature.json` 檔案，才能夠在 TREM-Lite 啟用擴充。
:::
